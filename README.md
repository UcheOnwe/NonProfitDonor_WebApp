# Nonprofit Donor Management Web App

A Blazor Server web application for managing donors, donations, and nonprofit records.  
Built using C#, SQL Server, Entity Framework Core, and Radzen components.

## Features
- Add Donors
- View Donors in a DataGrid
- SQL Server database with EF Core
- Automatically generated DB context and models
- Clean UI using Radzen components

## Technologies
- C# (.NET 8)
- Blazor Server
- Entity Framework Core
- SQL Server LocalDB
- HTML / CSS / Bootstrap
- Radzen UI

## How to Run
1. Clone the repo  
2. Restore NuGet packages  
3. Update the connection string in `appsettings.json`  
4. Run the application
